# Week1 - Prompt Engineering Experiments

## Goals

- Understand LLM parameters
- Learn Few-shot prompting
- Learn Chain of Thought
- Test structured output
- Explore prompt injection risks

## Experiments

### 1. Temperature Testing

Findings:

- High temperature increases creativity
- But reduces JSON stability

### 2. Few-shot Prompting

Findings:

- Significantly improves output consistency

### 3. Chain of Thought

Findings:

- Better reasoning
- Higher token cost

## Conclusion

Prompt engineering is less about wording tricks
and more about controlling model behavior.
